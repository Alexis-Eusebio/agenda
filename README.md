# agenda Flutter

Una aplicación de gestión de tareas que permite organizar actividades diarias mediante categorías, fechas de vencimiento, recordatorios y subtareas, manteniendo la persistencia de datos de forma local.
---

## Características Principales

* **Gestión Completa de Tareas:** Crea, edita y elimina tareas con campos específicos para notas y categorías.
* **Subtareas Dinámicas:** Desglosa tus objetivos en pasos más pequeños con estados independientes.
* **Calendario Visual:** Seguimiento de fechas mediante `table_calendar`, con indicadores de color según el estado de las tareas.
* **Persistencia de Datos:** Los datos no se pierden al cerrar la app gracias al uso de `shared_preferences`.
* **Modo Oscuro Personalizado:** Cambia entre tema claro y oscuro con un solo toque; la preferencia se guarda automáticamente.
* **Filtros Inteligentes:** Organiza tu vista por categorías como Trabajo, Personal, Lista de deseos y Cumpleaños.
